# Barrot Unified Ecosystem

## Overview
This repository contains the mutation-sealed Barrot ecosystem with:
- CI/CD pipelines for automation
- Guardian Rail for optimization
- Codex integration for cognition and orchestration
- Provenance ledger for integrity
- SSH Key Generator capability

## Key Features
- Automated rollback/recovery
- Bundle packaging with codex included
- Manual SSH key generation job
- Mutation-sealed provenance tracking

## Pipeline Stages
- **optimize**: Guardian Rail
- **merge**: Merge requests
- **export**: Bundle Drop Rail
- **provenance**: Ledger update
- **notify**: Status notifications

## Manual Jobs
- **ssh_keygen**: Generate SSH keys on demand

## How to Trigger Pipelines
- Push to `main` branch
- Schedule Guardian Rail via GitLab CI/CD
